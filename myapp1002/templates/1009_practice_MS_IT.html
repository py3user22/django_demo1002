#____ IaaS (Infrastructure as a Service)
----------------------
provides virtualized computing resources over the internet.
infrastructure relates to the physical machines on which the software runs.
Meaning to rent IaaS is to rent the high-spec computers on the cloud
ex1. Microsoft Azure, which provides Azure VMs for running apps,
.... Azure Blob storage, stores large amounts of unstructured data,
.... Azure Virtual networks which links different resources

#____ PaaS (Platform as a Service)
----------------------
platform for development, testing, deployment of applications
provides complete dev. env., including: OS, programming lang., dev tools.
ex1. Microsoft Azure PaaS
... Azure App Service --> provides web app hosting made in different programming languages.

#____ SaaS (Software as a Service)
----------------------
cloud computing that delivers software apps online
use of web browser to access the apps
software is hosted & managed by the cloud service provider
ex1. Azure SaaS --> Dynamics 365 --> comprehensive CRM (customer relationship mgmt) solution
