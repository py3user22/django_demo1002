#____ IDS/IPS types --> Intrusion detect system / Intrusion prevent system
----------------------
IDS --> 2 types:
HIDS --> host-based IDS (intrusion detect system)
|__> installed on a host or VM & monitors host activity,
|__> analyes the files, data, in/out traffic
|__> takes snapshot of the existing files in the system, & compares to a previous copy to compare against each other
.. if the shots differ, potential attack

NIDS --> network-based IDS
|__> monitors network traffic

IPS (intrusion prevent system)

#____ IPS types
----------------------
|.. includes AV (antivirus), anti-malware,
|.. constant monitoring of the system, and can prevent, report, block, drop it.

NIPS (network-based IPS) -->

WIPS (wireless IPS) --> monitors wifi network

HIPS (host-based IPS) --> installed at endpoints like PCs

NBA (network behavior analysis)

Proxy servers and types

#____ intermediary -- system or router that provides a gateway between users and internet
----------------------
can be setup as a filter or firewall
escentially a computer on the internet w/ a IP address of its own

types:
transparent proxy --> gives an identical exp. to what u would have if using home network

SOCK proxy --> a.k.a. socket --> operates at the TCP level & works by channeling your traffic thru a proxy server
.. which then passes the info to intended destination.

Anonymous proxy --> focuses on making internet activity untracable
.. access the internet on behalf of the user while hiding id & pc info.

Reverse proxy --> sits in front of clients web servers, and forwards request from a browser to web servers
.. intercepts requests from user at network edge of web browser.

VPN (virtual private networks)

#____ a service that protects your internet connection & online privacy
----------------------
creates a encrypted tunnel for your data, the key is known by your device
protects privacy by hiding the IP address of your device & allows to use public WiFi hotspots

VPN components:
VPN client
network access server
tunnel terminating device
VPN protocol

2 types:
Remote access VPN --> usually a tablet
site-2-site VPN --> typically used for branch offices to main branch

Core functions of VPN:
encapsulation
encryption
authentication

Virtualization types and info

#____ Types of virtualization
------------------
Virtualization -- process of creating a software based or virtual representation of something rather than physical.
allows to run multiple OS on a single physical machine.
sharing the resources of that single machine across the environments
ex1. apps, servers, storage, networks

Full virtualization
|__> allows multiple VM's to run on a host machine, each w/ own dedicated:
|___> OS, kernel, virtual hardware
|___> complete isolation between VM's

OS level virtualization
allows multiple instances of an app to run in isolated form of single OS.

Hybrid
separates resources or services from a physical delivery system w/in a hybrid cloud
Hybrid environments consist of 2 or more interconnected virtual, containers, private or public cloud environments

Hardware-assisted
enables full virtualization using help from hardware, primarily host processes
used to emulate a complete hardware env. or VM in which unmod'd guest OS using the
same instructions as the host machine runs in complete isolation w/ listed components
