------------------

//converting the previous if-else example with switch-case
switch(light) {

...... console.log("Get ready");

...... console.log("Don't drive");

Loops demo

#____ for loops --> continually executes repeated blocks of code until a certain condition is satified
------------------
i --> common counter used in JS
|__> used by loop to know when to start or stop

for (var i = 1; i<101; i++) {
.. console.log(i);
}

1| set the value --> var i = 0;
2| specify the condition --> i < 3;
3| increment the counter --> i++

#____ while loop --> will run as long as the condition returns true
|__> the counter is set outside the while loop / global variable
|__> the increment is done inside the loops body
-----------------
var i = 1;
while (i < 4) {
.. console.log(i);
.. i=i+1;
}

--------------------
for (var i = 10; i > 0; i--) {
.. console.log(i)
}
console.log('happy new years')

---------------------
var counter = 3;
while (counter > 0) {
.. console.log(counter);
.. counter = counter - 1;
}
console.log('happy day');
