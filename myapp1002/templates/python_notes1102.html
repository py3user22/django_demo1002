format specification in a replacement field "{}"

#___________________
name1 = "Pierre"
age1 = 7
name2 = "May"
age2 = 13

line1 = "{0:^7} {1:>3}".format(name1, age1)
line2 = "{0:^7} {1:>3}".format(name2, age2)
print(line1)
print(line2)

In the format strings in variable -> line1 and line2
argument index {0: ^7} # the 0 followed by colon :
format specifier {0: ^7} # the ^7 is the width of the field in the output

The width is preceded by a symbol <, >, ^
> for right aligned
< for left aligned
^ for center

# output
Pierre .....7
..May ....13

# optional --> attached to the format specifier --> numerical precision & type option
precision -> a number preceded by a period . & must appear after the alignment & width
type option -> single letter that always appears at the end of format specification

num1103 = 3.283663265
output = "{0:>10.3f} {0:.2f}".format(num1103)
print(output) #
