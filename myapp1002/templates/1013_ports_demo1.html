#_____ Ports
------------------
range: 0 - 65535
port authority: IANA (internet assigned numbers authority)
3 section ranges:
Well-known ports (0 - 1023)
|--> reserved for specific protocols & services used by system processes & apps
|--> reserved for the most commonly used & popular protocols in computer networks

Registered ports (1024 - 49151) -- used for specific services or protocols

Dynamic or private ports (49152 - 65535) -- used by client apps to connect to servers
|__> & are temp. assigned for the session duration
|__> when a pc has many open web pages, dynamic ports are used to identify different sessions

#_____ Firewall types
------------------
Packet filtering firewall -- inspect each data packet as it travels thru a network
Application-layer firewall -- target applications & monitor their behaviors
Circuit-lvl firewalls -- check weather TCP & UDP connections are valid before data is exchanged
proxy server firewalls -- server can monitor, filter, cache data request to/from a network
|__> client --> proxy server --> internet
stateful firewalls -- inspect connections of a network
|__> monitors all packets that go thru it, & stores info about the packets in a state table
|__> state table -- tracks sessions by recording port numbers as sessions start from inside the network
NGFW (next generation firewall) -- use of AI and ML to examine the payload for each packet
& inspect for suspicious characters or malware
