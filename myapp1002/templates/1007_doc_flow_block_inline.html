#___ block lvl element
---------------------------
will occupy the full horizontal width of its parent element & vertical height
each block element will have a new line b4 & after

< div> < /div> --> means divided element --> used to divide content into sections
< form> < /form>
< h1> < /h1> - < h5> < /h5>

#___ inline elements
---------------------------
< a href="index.html"> < /a>
< img src="photo.png">
< input>
< label> < /label>
< b> < /b> for bold
< i> < /i> for italics
< em> < /em> for emphasis
< span> < /span>

#___ box model intro
-------------

| margin --> seperates the element from neighbor elements
|---------
| border --> goes around the padding & content
|---------
| padding
|---------
| content --> actual content of the element

#___ padding css examples:
padding-top: 4px;
padding-right: 2px;
padding-bottom: 4px;
padding-left: 2px;

padding box width = content width + padding left + padding right

padding box height = content height + padding top + padding bottom

border types:
solid
dashed
dotted
double

border-width: thin
border-width: medium
border-width: thick

border box width = padding width + border left + border right

border box height = padding height + border top + border bottom

CSS rules: margin size
margin-top: 4px;
margin-bottom: 1px;
margin-left: 5px;
margin-right: 2px;
