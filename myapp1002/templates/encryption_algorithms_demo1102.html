DES (Data encryption standard

#_______
DES (data encryption standard)
breaks plain text input into 64 bit blocks to encrypt it using a cryptographic key

start -> 64 bit plain text
..|
..|___> IP (initial permutation) # splits into LPT--IP--RPT

LPT (left plain text)
..|
..|___> 16 rounds <---- key 64_bit --usage
RPT (right plain text)
..|
..|___> 16 rounds <---- key 64_bit --usage

==========================
64 bit key used to encrypt the blocks w/ 8 specific bits
..|
..|___> parity bits -> used to check for errors in the cipher text
..|___> every 8th bit is used for parity bit,
..|___> the key will have 56 bits after verification

==========================
LPT --- FP --- RPT conversion
FP (final permutation)
..|
..|
..|---> 64 bit cypher text

#_______ AES (Advanced encryption standard)

#_______
Symmetric encryption
divides data into blocks of equal size, ex.. 128 bits
starts with plain text
....|
....|____> Round 1:
....|____> byte substitution
....|____> Shifting of rows to the left
....|____> Mixing of columns
....|____> Add another round key

_________________________
Last round of mixing
....|____> byte substitution
....|____> byte substitution
....|____> Add another round key

After all rounds are completed, the final block is made into cipher text
Note: You will need the correct key to see the data


==========================
Steps are repeated for a number of rounds
128 bit keys go for 10 rounds

192 bit keys go for 12 rounds

256 bit keys go for 14 rounds
