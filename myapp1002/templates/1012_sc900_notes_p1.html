#____ Active Directory
------------------
directory --> hierarchical structure that stores info about objects on the network
directory service ---> stores directory data & makes it avail. to network users, admins, services & apps

AD (Active Directory) --> is a set of directory services for on-prem domain-based networks
AD DS (Active directory domain services)
--> stores info about members of the domain,
--> devices and users
--> verifies credentials
--> defines the access rights
--> Domain Controller (DC) --> server running the AD DS
--> central component in org's w/ on-prem IT infrastructure
--> gives orgs the ability to manage multiple on-prem infrastructure components & systems using a single identity per user
