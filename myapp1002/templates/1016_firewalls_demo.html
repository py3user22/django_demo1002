#____ firewall types:
--------------------
Host-based --> software based firewall, only 1 device

Network-based
--> hardware based firewall used to filter in/out traffic from a LAN
--> can be hardware or added software included w/ router

Packet filtering
type that controls data flow in/out of a network
use of user-defined IP addresses, protocols, ports, rules
OSI model: layer 2 data link - layer 3 network - layer 4 transport

Circuit Level gateway
provides connection security to internal/external pc in a networks session layer
TCP/IP model ---> works between the transport & application layers
OSI model -----> session -layer 5
relays TCP connections, & doesnt do packet filter based on packet content

Application level gateway a.k.a. proxy firewall
OSI model ---> applicaion layer 7
filters traffic based on data packet content rather than just packet headers
ex1. URL, filtering, content filtering, authN

Application proxy firewall
filters messages at the application
considered the most secure form as no networks actually contact the systems
has own internal protocol address, so no ext. network connection can get packets directly from the network
OSI model: layer 7 Application

SMLI (Stateful multilayer inspection) firewall
uses high tech form of packet filtering
examines all 7 layers of OSI model
examines the entire packet, including the data
OSI model: layer 3 Network

NAT (network address translation) firewall
uses NAT to protect network from unauthorized access
private network devices IP addresses are hidden when communicating w/ internet
OSI model: layer 3 network

VPN firewall
designed specifically to protect intercepts of VPN connection
OSI model: layer 2 -7
