#___ operators in JS
---------------
Logical AND operator: && --> used to confirm if multiple comparisons are true

Logical OR operator: || --> when to chk if at least one of given comparisons is true

Logical NOT operator: ! --> think of as switch, flips the boolean value from T to F or F to T

The modulus operator: %

The equality operator: ==

The strict equality operator: ===

The inequality operator: !=

The strict inequality operator: !==

The addition assignment operator: +=

The concatenation assignment operator: += (it's the same as the previous one - more on that later)

ex2. check if time is between certain numbers using AND &&
-------------------
var currentTime = 10;
console.log(currentTime > 9 && currentTime < 17);

ex3. check if time is either or ||
---------------
var currentTime = 7;
console.log(currentTime < 9 || currentTime > 17); // either x<9 or x>17

ex4. check using the NOT ! operator
---------------------
var petHungry = true;
console.log('Fed the pet some food')
console.log('Pet is hungry: ', !petHungry); // Pet is hungry: false
