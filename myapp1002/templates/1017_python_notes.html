not booleans

#____ not (p or not q)
----------------------
1) p is True, q is True
|--> not (T or not T) --> naughty naughty
|--> not true or false --> F

2) p is True, q is False
|--> not (T or not F) -->
|--> not true or true --> False
3) p is False, q is True -->
|--> not (F or not T)
|--> not false or false --> True
4) p is False, q is False
|--> not (F or not F)
|--> not false or true --> False
