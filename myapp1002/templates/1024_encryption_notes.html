#____ Cyptography techniques
--------------------
Encryption
----------------
converts plain text into cyphertext, which can only be decoded by the intended receiver
cipher --> mathematical formula that transforms plaintext into ciphertext

Symmetric
uses the same key to encrypt & decrypt data.
both sender/recipient have identical copies of key.
Cipher text --> data is encrypted into cypher text, meaning no1 can read or understand w/o key

Asymmetric
-- aka public key cryptography-------------
type of encryption that uses a pair of keys to encrypt/decrypt data
sender uses the recipients public key to encrypt the data
public key is available to anyone
only the private key is used to unlock the messages
recipient uses the private key to decrypt it

eliminates the need to exchange secret keys

DES (date encryption standard)

#____ types: DES (symmetric)
breaks plain text input into 64 bit blocks to encrypt
starts w/ IP (initial permutation)
|_> splitting the block into LPT (left plain text) | IP | RPT (right plain text)
|__> each having 16 rounds of encryption --> LPT @ 16 + RPT @ 16
|___> rejoining the FP (final permutaion) to make the 64 bit cipher text

DES uses a 64 bit key to encrypt these block w/ each 8 bits called parity bits used to check for errors in cypher text

AES (advanced encryption standard)

#____ AES (symmetric)
divides data into block of equal size, 128 bits
1st round: involves byte substitution, shifting rows cyclically to the left mixed columns, adding another round
key steps 4-7 are repeated for a # of rounds
10 rounds for 128 bit keys
12 rounds for 192 bit keys
14 rounds for 256 bit keys

after the rounds are completed, the final block is produced as cypher text
so if get hold of the cipher text, u wont decrypt it w/o the correct key

RC4 encryption algorithm

#____ is a stream cipher & variable length key algorithm
that encrypts one bit at a time.
using a pseudorandom bit generator to produce an unpredictable 8 bit number
combined w/ plain text stream cipher using XOR operation

RC5 (symmetric)
-----------------
known for simplicity, speed, memory consumption
addresses 2 word blocks at a time
can be defined based on input, plain text, block size, # of rounds, key size

RC6 ( symmetric )
----------------

DSA (digital signature algorithm)

#____ public key crypto system used for digital signatures
system involves key generation where sender keeps their private key secret
& signature generation
----------------
sender creates a digital signature using their private key
the receiver then verifies the messages authenticity using senders public key

provides message authN, integrity, & non-repudiation
ensuring message was sent by private key holder

private key is used by the sender to generate a unique digital signature

public key is used by recipient to verify the authN of signature
